How To Use - General User

ChickenBus operates on a web-based portal, and a mobile application. In order to access bus route information simply enter your current location and your desired location in the search panel.

User must select the suggested location for origin and destination from the auto-complete drop down. Optimal routes will appear next the search panel in the map interface.

How To Use - Admin

Login enables more functionalities, including "Route Entry" and "Data Interface"

Route Entry

In route entry, admin can name the route, add the stops, add time, duration, cost, and notes

Or to bulk entry a formated csv file with the correct fields

Then the admin can preview or add the route

Data Interface

This interface allows the admin to access the database info without external programs It gives the admin the option to modify routes, stops, and users

Modification includes delete and approve

How To Use - Mobile

Interface is same for all level of user

Home tab contains our search function with the same auto-complete origin and destination

By clicking "search" the user is taken to the Map tab, and the origin, destination are pinned and drawn

The route info is hidden in the pin of destination, and can be accessed by clicking on it

FAQS

  • What if nothing is returned when I enter my origin, destination, and departure time? - Then it is likely there is not a route within your specified radius. It is equally likely that we do not have any bus route information for your current area.
  • Route Entry is equiped with warning to catch any invalid input
  • Bulk import must be formated exactly as stated, see admin.md
