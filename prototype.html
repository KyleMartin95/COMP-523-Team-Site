The prototype is as follows: Deployable Google maps reminiscent web portal with a mobile app converted environment to easily both store and access bus route information that is otherwise unregistered.

This includes the following functionalities:

  • search entry page to access routes
  • data entry page to store and collect previously unregistered routes
  • backend to access entered routes through the search entry page
  • routing algorithm to find most relevent and time efficient route
  • robust UX for simple usability
